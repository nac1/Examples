# Examples
A complete set of examples for the eight types of norms is provided for each of the two approaches.

- For the **Controlled Natural Language (CNL)** approach, we use Attempto Controlled English (ACE).
- For the **Large Language Model (LLM)** approach, we use ChatGPT.

Both techniques accept **prompts** in TXT format as input and generate **ontologies in OWL 2** as output.

The repository includes three folders with the following contents:  
1. **prompts**: Contains the prompts described in the article
2. **CNL_examples**: Contains eight norms and the ontologies using ACE grammar.
3. **LLM_examples**: Contains eight norms and the ontologies using a LLM.
